# 项目文件夹

偶魔的采用 Omoospace 方案管理项目文件夹。在它基础上设置了如下结构：

```bash
[root-dir]
├── .obsidian          # obsidian 配置文件
├── contents/          # 被引用的资源、成品
├── notes/             # 记录文档
├── references/        # 资料文件
├── subspaces/         # 工程文件、制作文件
├── OMOOSPACE.md       # Omoospace
└── README.md          # 自述文档
```


存放和项目相关的，项目专属的文件，例如
- 项目专属的配置文件
- 创作过程文件

私密的、涉及安全的，绝不能放入，例如
- 密钥、环境配置
- 合同、报价单
